Órdenes i

Ejecutar Órdenes

Envía órdenes de compra o venta simuladas al broker activo. Yahoo Finance no ejecuta órdenes reales. Binance e Interactive Brokers tienen placeholders para integración real con HMAC e ib_insync.

↗ Investopedia: Order Types

Análisis Técnico i

Análisis Técnico

Evalúa acciones usando indicadores matemáticos sobre datos históricos de precio y volumen. Scalping para corto plazo (minutos) con EMA, RSI y Bollinger. Swing para largo plazo (días) con MACD, SMA 200 y Soportes/Resistencias.

↗ Investopedia: Technical Analysis

🤖 Asistente de Trading i

Análisis Automático con IA

Evalúa indicadores técnicos en tiempo real usando reglas de mercado. Genera recomendaciones de compra, venta o espera con nivel de confianza y razones detalladas. Ideal para aprender por qué se toma cada decisión.

Seleccioná un ticker y ejecutá "Analizar" para ver el análisis del asistente debajo del gráfico.

Leyenda de Estrategias

Scalping (1m-5m)
  • EMA 9/21 crossover → ±35% confianza
  • RSI 14 (sobreventa <30 / sobrecompra >70) → ±25%
  • Bollinger Bands (20,2) → ±20%
Swing (1d)
  • MACD crossover → ±40% confianza
  • SMA 200 → ±15% tendencia
  • Soporte/Resistencia histórico → ±20%
